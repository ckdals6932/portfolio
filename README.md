## 📋 Study Rule
### 📌 주 5회 [주말 제외]
1. 당일 문제를 해결한다.
2. 전 날 코드에 주석을 달아 피드백 해준다.
3. 그저께 코드에 달린 피드백을 반영한다.
4. 내일 풀 문제를 정한다.
